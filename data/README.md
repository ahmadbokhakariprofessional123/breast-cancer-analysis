# Dataset Information

This project uses the **Breast Cancer Wisconsin (Diagnostic) Dataset**.

## Source
- UCI Machine Learning Repository  
- Link: https://archive.ics.uci.edu/ml/datasets/Breast+Cancer+Wisconsin+(Diagnostic)

## Dataset Details
- Total samples: 569
- Number of features: 30 numeric features
- Target variable: Diagnosis
  - 0 → Benign
  - 1 → Malignant

The features are computed from digitized images of fine needle aspirates (FNA) of breast masses and describe characteristics of the cell nuclei such as radius, texture, smoothness, concavity, and symmetry.
